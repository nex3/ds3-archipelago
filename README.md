## Dark Souls III Archipelago Client

The Dark Souls III client for the Archipelago multiworld randomizer. See
[archipelago.gg] for general information about Archipelago, and [the Dark Souls
III setup guide] for instructions on setting up and using this mod.

[archipelago.gg]: https://archipelago.gg/
[the Dark Souls III setup guide]: https://nex-3.com/ds3/setup
